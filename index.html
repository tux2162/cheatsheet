GIT


Remonter
git init: pour créer le fichier .git
git add . : pour ajouter dans le stage
git commit -m " : pour commiter les éléments du stage"
git push : pour envoyer sur le dépôt
git pull : pour se synchroniser au dépôt
git branch : liste les branches (en local)
git checkout -b : créer une branche et switch de branche vers la nouvelle branche apres la création (en local) une fois le push la branche sera envoyé sur le dépôt
git checkout : pour changer de branche

LINUX


Remonter
Rendre un exécutable utilisable par tout les utilisateurs : mv /usr/local/bin/ -v
Ensemble des répertoires où se trouvent les programmes exécutables : echo $PATH
Créer un fichier de 1000M : dd if=/dev/zero of=fic bs=1M count=1000 | ls -l -h fic | sync
Voir la vitesse d'un disque: hdparm -t
Afficher les manuels qui contiennent un mot-clef : man -k
Formater et créer un système de fichiers : mkfs
Afficher les partitions : lsblk -f
Afficher les partitions : lsblk -f
generer une clé ssh : ssh-keygen
envoyer la clé sur le serveur : ssh-copy-id @

VIM


Remonter
:x = :wq
G : retourner en bas du fichier
gg : retourner en haut du fichcier / : chercher un mot n pour aller en avant N précédent
$ : pour aller en bout de ligne
o : pour aller à la ligne et passer en insertion
y : pour copier
c : pour couper
p : pour coller
